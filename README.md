# Sample reactjs app, which consumes birdio graphql api (https://github.com/xurshid29/birdio-graphql-api)

## Installation

1. Duplicate .env.dist file and rename it as .env
2. Modify environment variables (inside .env)

Then:

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# production 
$ npm run build
